## Overview
This project recreates the classic Snake arcade game using Python. The player navigates a snake inside a grid, collecting randomly spawned food items. Each food eaten increases the snake's length and score. But be careful‚Äîrunning into the walls or your own tail will end the game instantly.

---

## Features
- Classic snake movement controls (Up, Down, Left, Right)
- Random food spawning
- Real-time score tracking
- Collision and game-over detection
- Increasing difficulty as the snake grows
- Clean restart functionality
- Smooth Tkinter-based GUI

---

## Tech Stack
- **Language:** Python  
- **GUI Library:** Tkinter  
- **Modules Used:** `random`, `time`

---

## How to Play
1. Run the game (`snake-game.py`)
2. Use arrow keys to control the snake
3. Eat food to score points and grow longer
4. Avoid:
   - Screen borders
   - Your own tail
5. Game ends on collision
6. Restart to challenge your high score!
